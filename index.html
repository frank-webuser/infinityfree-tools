Cloudflare with free subdomain

You must own a custom domain to use Cloudflare. You cannot use a free subdomain. If you have a custom domain, you can connect it to Cloudflare via dash.cloudflare.com and then create (in the DNS section on CF) the correct A records with the IP address of your hosting account.

How to name files/folders

![image|690x318](upload://3PheMfAO5m1Y1GTMgpPZWP9Lo9U.png) ### How to Name Files/folders Do Not Use Special Characters For best results, you should only use the letters a-z, A-Z, the numbers 0-9, hyphens (-), underscores (_) and periods (.). Any other character could get you in trouble, causing the file not to load or the page to load incorrectly. it is best to use lowercase letters, also Do Not Use Any Spaces.

Enable Directory Listing

Please create a .htaccess file (on root) with this content: ``` #Disable Directory Listings in this Directory and Subdirectories Options +Indexes ```

Forcing HTTPS

Please create a .htaccess file (on root) with this content: ``` RewriteEngine On RewriteCond %{HTTP:X-Forwarded-Proto} !https RewriteCond %{HTTPS} off RewriteCond %{HTTP:CF-Visitor} !{"scheme":"https"} RewriteRule (.*) https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301] ```
